# Blogger テンプレート

QooQ をベースにカスタマイズした Blogger 用テンプレートです。

---

## 特徴

- **ヘッダー横並びレイアウト** — タイトル左・グローバルメニュー＋検索右
- **プルダウンメニュー** — メニュー名の先頭に `_` をつけると子メニューになります
- **ページネーション** — 記事一覧・ラベル・アーカイブページ対応
- **サイドバー目次** — 個別記事ページでスクロール追従
- **人気記事 / 新着記事タブ** — タブ切り替えで表示
- **関連記事** — 記事下部に自動表示
- **SNSシェアボタン** — Twitter / Facebook / はてなブックマーク / Pocket / LINE
- **パンくずリスト** — 構造化データ（Schema.org）対応
- **レスポンシブ対応** — スマホはハンバーガーメニュー
- **カラーカスタマイズ** — テンプレートデザイナーからブランドカラーを変更可能
- **FontAwesome 6** 採用

## 使い方

### 1. テンプレートの適用

1. Blogger 管理画面 → **テーマ** → **HTMLを編集**
2. 既存のコードをすべて削除し、`template.xml` の内容を貼り付けて保存

### 2. 必須設定（適用後に変更してください）

#### ブログURL（2箇所）

サイドバーの「人気記事 / 新着記事」ウィジェット内に以下が2箇所あります。

```javascript
const blogUrl = "https://your-blog.blogspot.com";
```

`your-blog.blogspot.com` をご自身のブログのURLに変更してください。

> Blogger 管理画面 → テーマ → HTMLを編集 → `your-blog.blogspot.com` で検索

#### ファビコン

head 内に以下があります。ご自身のファビコン画像URLに変更してください。

```html
<link href='★ファビコンURL' rel='icon' type='image/x-icon'/>
```

不要な場合はこの行ごと削除してください。

#### OGP デフォルト画像

SNSシェア時のデフォルトサムネイルです。ご自身の画像URLに変更してください。

```html
<meta content='★画像URL' property='og:image'/>
```

#### プロフィール画像

サイドバーのプロフィールウィジェットを直接編集してください。

- カバー画像：`/* ★カバー画像URLをここに設定してください */` の箇所
- アバター画像：`src="https://placehold.jp/120x120.png?text=Avatar"` の箇所

### 3. プルダウンメニューの使い方

Blogger の「ページリスト」ウィジェットでメニューを設定する際、**子メニューにしたい項目のタイトルを `_`（アンダースコア）から始める**と自動的にプルダウンになります。

```
ホーム          → 通常メニュー
カテゴリ        → 通常メニュー（親になる）
_サブメニューA  → カテゴリの子メニュー
_サブメニューB  → カテゴリの子メニュー
お問い合わせ    → 通常メニュー
```

### 4. 記事一覧の表示件数変更

`nextprev` 内の `maxResults` の値を変更してください（デフォルト: 4）。

```javascript
const maxResults = 4; // ← この数字を変える
```

### 5. 目次の出し方

目次を表示させたい記事の該当箇所に以下を記述してください。

```html
<div id="toc"></div>
```

見出し（h2・h3・h4）が2つ以上ある記事では、この位置に自動で目次が生成されます。

---

### 吹き出しショートコードの使い方

記事エディタの **HTML モード** で以下のショートコードを貼り付けるだけで、
会話風の吹き出しが表示されます。

---

#### 基本の書き方

#### 左の吹き出し

```html
[chat-left img="画像URL" name="名前"]セリフ[/chat-left]
```

#### 右の吹き出し

```html
[chat-right img="画像URL" name="名前"]セリフ[/chat-right]
```

---

### 使用例

```html
[chat-left img="https://example.com/panda.jpg" name="パンダくん"]おはよう！[/chat-left]

[chat-right img="https://example.com/person.png" name="太郎"]おはよう！元気？[/chat-right]

[chat-left img="https://example.com/panda.jpg" name="パンダくん"]元気だよ〜！[/chat-left]
```

---

### 画像なしで使う場合

`img` と `name` は省略できます。その場合はグレーのプレースホルダー画像が表示されます。

```html
[chat-left]おはよう！[/chat-left]

[chat-right name="太郎"]やあ！[/chat-right]
```

## カスタマイズ

テンプレートの CSS 内（`吹き出しショートコード` のセクション）を編集することで
色・サイズ・形を変更できます。

| CSS クラス | 対象 |
|---|---|
| `.serif-left` の `background` | 左吹き出しの背景色 |
| `.serif-right` の `background` | 右吹き出しの背景色 |
| `.icon-left img` の `border` | 左アイコンの枠色 |
| `.icon-right img` の `border` | 右アイコンの枠色 |
| `border-radius: 50%` | アイコンを円形に（四角にしたい場合は `0` に） |
| `border-radius: 10px` | 吹き出しの角丸 |

## よくある質問

**Q. 画像はどこにアップロードすればいいですか？**
Blogger の「画像」ウィジェットや Google フォト、Imgur などにアップロードして URL を取得してください。

**Q. セリフに改行を入れたいです**
セリフ内で `<br>` タグが使えます。
```html
[chat-left img="..." name="パンダくん"]おはよう！<br>今日もいい天気だね。[/chat-left]
```

## カスタマイズ

### ブランドカラーの変更

Blogger 管理画面 → テーマ → カスタマイズ から以下の色を変更できます。

| 変数 | 説明 | デフォルト |
|------|------|-----------|
| ブランドカラー | ヘッダー背景・見出し背景など | `#0f2b46` |
| ブランドカラー（文字） | ヘッダー内の文字色 | `#ffffff` |
| ブランドサブカラー | スマホメニュー背景など | `#008db7` |
| 最背景色 | ページ全体の背景 | `#f6f5f5` |
